# Trash Disposal Telegram Bot

This is a simple Telegram bot designed to help track trash disposal responsibilities among a group of people. The bot allows users to check who is responsible for taking out the trash and mark when the trash has been taken out for the day.

## Features
- Check who is responsible for taking out the trash today.
- Mark the trash as taken out.
- Reset the trash status for the next day.

## Prerequisites

Before running the bot, ensure you have the following Python modules installed:

- `python-telegram-bot`

You can install the required module using pip:
```bash
pip install python-telegram-bot
```

## How to Run

1. Clone the repository to your local machine.
2. Navigate to the project directory.
3. Run the bot using Python:
   ```bash
   python python_bot.py
   ```

## Usage

- Use the `/start` command to begin interacting with the bot.
- Use the buttons provided by the bot to check responsibilities and mark trash disposal.
- Use the `/next` command to reset the trash status for the next day.

## Note

Ensure you have a valid Telegram bot token and update the bot configuration accordingly.
